# Task6. Ограничение количества запросов в Nginx

В файле `nginx.conf` настроено ограничение не более 10 запросов в минуту по адресу клиента. При превышении лимита Nginx возвращает код 429.

Лимит держит зона `partner_limit` с правилом `rate=10r/m`. Ключом выбран адрес клиента, на реальном API партнёра удобнее ключевать по идентификатору из заголовка авторизации.
